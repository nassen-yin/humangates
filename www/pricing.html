计费说明

Human Gates 的收费模式，简单透明。

API 调用免费

创建任务、查询状态、查看时间线等所有 API 调用不收费。仅在任务执行成功后收取服务费。

服务费含什么

含资料审核、服务商匹配、进度跟踪、结果回传。不包含政府规费（刻章费、工本费），实报实销。

失败处理

因政府政策变动或不可抗力导致无法办理的，全额退还服务费。因资料造假导致被驳回的，酌情扣除审核成本。

开发者集成

SDK、MCP Server、Webhook 集成支持正在开发中。现阶段通过 REST API 直接调用即可。
